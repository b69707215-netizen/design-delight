# Gross Academy content format

Эта папка фиксирует структуру контента для стабильного импорта из GitHub.

## Файлы

- `content/programs.json` — список программ обучения.
- `content/cards.json` — карточки контента, связанные с программами или отдельными материалами.
- `content/cards/*.mdx` — полный текст карточек, если карточке нужен длинный материал.
- `content/schemas/*.schema.json` — JSON Schema для валидации формата.

## Языки

Все публичные тексты должны иметь две версии:

```json
{ "uk": "Український текст", "en": "English text" }
```

## Правила стабильного импорта

1. `slug` не менять после публикации — он используется как постоянный идентификатор.
2. Для программ `amount` хранится числом в минимальной валютной единице сайта, сейчас UAH.
3. `lessonsCount` — количество купленных/входящих уроков для оплаты и кабинета.
4. Не удаляйте поля `version` и `updatedAt`.
5. Новые поля сначала добавляйте в `schemas`, затем в JSON.
6. Неактивные элементы оставляйте с `isActive: false`, а не удаляйте.

## Ожидаемые пути для импорта

- Programs: `content/programs.json`
- Cards: `content/cards.json`
